# DDoS Attack Detection and Analysis

## Overview

This project focuses on detecting and analyzing **Distributed Denial of Service (DDoS) attacks** using data analysis and machine learning techniques.

DDoS attacks attempt to overwhelm a system, server, or network by flooding it with a large volume of traffic, making services unavailable to legitimate users. This project analyzes network traffic patterns to identify potential DDoS attack behavior.

## How to Run the Project

### 1. Clone the repository

git clone https://github.com/YOUR_USERNAME/DDoS-attack-detection.git

### 2. Navigate to the project directory

cd DDoS-attack-detection

### 3. Install required dependencies

pip install -r requirements.txt

### 4. Launch Jupyter Notebook

jupyter notebook

### 5. Open the notebook

DDoS_Analysis_Final.ipynb

Run all the cells to execute the analysis.
